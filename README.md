# CppPractice
Este repositorio contiene una colección de ejercicios y problemas desarrollados en C++ 


 
## Objetivos  
- Mejorar las habilidades de programación en C++, incluyendo la estructura de un programa, el manejo de tipos de datos y estructuras de control.  
- Desarrollar habilidades algorítmicas mediante la solución de problemas clásicos (cálculo de residuo, factorial, números primos, series, etc.).  
- Implementar ejercicios que van desde operaciones aritméticas simples hasta problemas más complejos como la generación de espirales, la serie de Collatz y la aproximación de constantes matemáticas.

## Ejercicios y Problemas Incluidos

1. **Residuo de una División:**  
   - Programa que pide dos números A y B e imprime el residuo de la división A/B.

2. **Par o Impar:**  
   - Programa que determina si un número N es par o impar.

3. **Número Mayor:**  
   - Dado dos números A y B, se imprime el mayor.

4. **Número Menor:**  
   - Dado dos números A y B, se imprime el menor.

5. **División Redondeada:**  
   - Programa que realiza la división A/B y muestra el resultado redondeado.

6. **Potencia sin Librerías:**  
   - Calcula \( A^B \) sin usar librerías matemáticas.

7. **Sumatoria:**  
   - Suma todos los números desde 0 hasta N.

8. **Factorial:**  
   - Calcula el factorial de N.

9. **Círculo:**  
   - Calcula el perímetro y el área de un círculo con radio N (usando 3.1416 como aproximación de π).

10. **Múltiplos:**  
    - Imprime todos los múltiplos de N entre 1 y 100.

11. **Tabla de Multiplicar:**  
    - Muestra la tabla de multiplicar de un número N hasta 10.

12. **Potencias de un Número:**  
    - Imprime las potencias de N desde \( N^1 \) hasta \( N^5 \).

13. **Divisores de un Número:**  
    - Lista todos los divisores de N.

14. **Columnas Paralelas:**  
    - Imprime dos columnas paralelas: de 1 a 50 y de 50 a 1.

15. **Sumatoria Acumulativa (hasta cero):**  
    - Suma números ingresados constantemente hasta que se ingrese 0.

16. **Promedio Acumulativo:**  
    - Calcula el promedio de números ingresados (excluyendo el cero) hasta que se ingrese 0.

17. **Número Mayor en una Serie:**  
    - Determina el mayor de una serie de números ingresados hasta que se ingrese 0.

18. **Cuadrado Perfecto:**  
    - Verifica si un número N es un cuadrado perfecto.

19. **Número Primo:**  
    - Determina si un número N es primo.

20. **Número Palíndromo:**  
    - Verifica si un número es palíndromo (se lee igual de izquierda a derecha y viceversa).

21. **Conversión de Caracteres:**  
    - Recibe un carácter y, si es letra, lo convierte de mayúscula a minúscula o viceversa.

22. **Formato de Tiempo:**  
    - Convierte una cantidad de segundos a formato horas:minutos:segundos.

23. **Mínimo Común Múltiplo (MCM):**  
    - Calcula el MCM de dos números A y B.

24. **Cuadrado con Bordes:**  
    - Imprime un cuadrado de tamaño N con bordes dibujados con el carácter '+' y el interior vacío.

25. **Cantidad de Dígitos:**  
    - Determina cuántos dígitos tiene un número N.

26. **Tipo de Triángulo:**  
    - Recibe tres números y determina si forman un triángulo (isósceles, equilátero, escaleno) o si no forman uno.

27. **Calculadora Básica:**  
    - Opera como calculadora realizando suma, resta, multiplicación y división.

28. **Aproximación de Pi:**  
    - Calcula una aproximación de π mediante una suma infinita con un número de términos determinado por el usuario.

29. **Juego de Adivinanza (Manual):**  
    - El programa intenta adivinar un número seleccionado por el usuario (el usuario solo indica si es mayor, menor o igual).

30. **Juego de Adivinanza (Aleatorio):**  
    - Se genera un número aleatorio y el usuario debe adivinarlo, recibiendo pistas de mayor o menor hasta acertar.

31. **Identificación de Vocales/Consonantes:**  
    - Determina si un carácter ingresado es una vocal, una consonante o no es una letra.

32. **Cambio de Billetes y Monedas:**  
    - Calcula la mínima combinación de billetes y monedas para una cantidad de dinero determinada.

33. **Validación de Fecha:**  
    - Verifica la validez de una fecha (con especial manejo del 29 de febrero).

34. **Suma de Horas:**  
    - Suma una hora y una duración dadas en formato 24 horas.

35. **Patrón en Pantalla:**  
    - Imprime un patrón en forma de diente de sierra (con tamaño determinado por un número impar).

36. **Aproximación del Número de Euler (e):**  
    - Aproxima el valor de e usando la suma infinita de sus términos factoriales.

37. **Fibonacci y Suma de Pares:**  
    - Calcula la suma de los números pares en la serie de Fibonacci menores a un valor n.

38. **Suma de Múltiplos Combinados:**  
    - Suma los múltiplos de dos números (sin duplicar los comunes) menores a un tercer número.

39. **Suma de Dígitos Elevados a Sí Mismos:**  
    - Suma cada dígito de un número N elevado a sí mismo.

40. **Enésimo Número Primo:**  
    - Encuentra el enésimo número primo.
    
41. **Mínimo Común Múltiplo de 1 a N:**  
    - Calcula el MCM de todos los números enteros entre 1 y N.

42. **Máximo Factor Primo:**  
    - Encuentra el mayor factor primo de un número.

43. **Suma de Primos Menores a N:**  
    - Suma todos los números primos menores a N.

44. **Palíndromo Más Grande de 3 Dígitos:**  
    - Encuentra el número palíndromo más grande obtenido como multiplicación de dos números de 3 dígitos.

45. **Espiral Numérica:**  
    - Genera una espiral numérica en una matriz de tamaño n×n y calcula la suma de sus números en la diagonal.

46. **Serie de Collatz:**  
    - Encuentra, para un número k dado, la semilla menor que k que produce la serie de Collatz más larga, e imprime dicha serie.

47. **Números Triangulares y Divisores:**  
    - Determina el primer número triangular que tiene más de k divisores.

## Cómo Utilizar Este Repositorio

1. **Clonar el repositorio:**  
   ```bash
   git clone <URL_del_repositorio>
   ```
2. **Abrir en QtCreator o cualquier otro IDE que compile c++:**  
   Cada ejercicio se encuentra en su propio proyecto de consola. Puedes abrirlos en QtCreator para compilar, ejecutar y depurar el código.

3. **Compilar y Ejecutar:**  
   Sigue las instrucciones del IDE para compilar y ejecutar cada proyecto. Revisa los comentarios en el código para entender la lógica implementada en cada ejercicio.

## Consideraciones


- **Formato de Salida:**  
  Algunos ejercicios tienen especificaciones en el formato de salida (por ejemplo, el manejo de acentos y tildes en la terminal). Se ha seguido la guía proporcionada para mantener la consistencia en la salida de los programas.
